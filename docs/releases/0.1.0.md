# GlyphsIconGrid 0.1.0

The first public release adds a weight-aware construction grid to Glyphs 3.5 and Glyphs 4.

- Use one `IconGrid.gridSize` per master for both square-cell size and circular spacing.
- Start with 34 units on Regular and 72 units on Bold for a 1000-UPM icon family.
- Use the cell-centered `odd` grid by default, or opt into line-centered `even` mode.
- Draw square grids, concentric circles, spokes, live-area frames, and common icon keylines behind the active glyph.
- See strict non-snapping alignment feedback while drawing or moving nodes.
- Inspect and change settings safely through the included Glyphs MCP skill.

See the [user guide](https://github.com/thierryc/GlyphsIconGrid/blob/v0.1.0/docs/USER_GUIDE.md) and [complete parameter reference](https://github.com/thierryc/GlyphsIconGrid/blob/v0.1.0/docs/PARAMETERS.md).
